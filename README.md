# Network Diagnostics Toolkit

A Python-based Network Diagnostics Toolkit for network monitoring and troubleshooting.

## Features

* Ping Utility (ICMP)
* Traceroute (TTL-based route discovery)
* Packet Analyzer using Scapy
* Protocol Detection (TCP, UDP, ICMP)
* Hostname Resolution
* CSV Packet Logging
* TCP Port Scanner
* Service Detection (HTTP, HTTPS, SSH, DNS, etc.)

## Technologies Used

* Python
* Scapy
* Socket Programming
* CSV Logging
* Git & GitHub

## Concepts Covered

* TCP
* UDP
* ICMP
* TTL
* DNS Resolution
* Routing
* Packet Sniffing
* Port Scanning
* Network Troubleshooting
